Detailed Description

The ComboBox is an input control that supports user to either select a value from predefined value list or type in any other value.

The ComboBox control is a combination of a list box and an edit control. The end user may select a value from the predefined value list or type in a value through the edit control. And the control will send the value out through output parameter to trigger any further action(s).

The ComboBox node contains following child nodes,

  • All the nodes defined in ViewElements.
  • [Optional] The prompt node in type of Prompt defines the prompt for the ComboBox input.
  • [Optional] The options node in type of OptionList defines the preset values that the end user can select one from them.
