Enumeration LoadDataAt

Description

This is the enumeration type for view attribute loadDataAt. It indicates when to load data for a view control. The supported values are,

  • Default - Data for the view will be loaded when there is change in the input parameters.
  • Creating - Data for the view will be loaded when the view is created.
  • Visualizing - Data for the view will be loaded when the view becomes visible first time.

Supported controls:

  • ChartView
  • ListView
  • TextView
  • TreeListView
  • TreeView
  • WebView
  • Button
  • ComboBox
  • PropertyInput
  • TextInput
