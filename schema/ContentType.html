Enumeration ContentType

Description

This is the enumeration type for attribute contentType that is used in WebView. It indicates the content type returned from the data source defined for the control. The supported values are,

  • Url - The page should be loaded from the URL returned through data source.
  • Content - The page should be loaded from the raw content returned from data source.
  • File - The page should be loaded from the files defined.

Supported controls:

  • WebView
